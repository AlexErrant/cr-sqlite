# P2P TODO MVC

Usage:

Start the "peer server"
```
cd ../peer-server
pnpm install
pnpm start
```

Start the app:
```
pnpm install
pnpm start
```

Open instances _in different browsers_ (i.e., safari, firefox, chrome) and connect between them.

Or, open them all in the same browser which uses cross-tabs comms.
